# Frontend приложения обмена ообщениями
* Vue 3 Composition Api
* Vue-Router
* Pinia

 В файле .env проверить адрес подключения к серверу

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
